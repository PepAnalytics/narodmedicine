# Народная Медицина — Мобильное приложение

Flutter-приложение для поиска народных методов лечения по симптомам.

## 📱 Возможности

- **Поиск по симптомам** — введите симптомы и найдите подходящие заболевания
- **Просмотр заболеваний** — подробное описание болезней и симптомов
- **Методы лечения** — народные рецепты с ингредиентами и инструкциями
- **Уровень доказательности** — оценка эффективности методов (высокий/средний/низкий)
- **Озвучка рецептов** — текстовое воспроизведение рецептов (TTS)
- **Оценка методов** — лайки, дизлайки и отзывы
- **Темная тема** — автоматическое переключение светлой/темной темы

## 🚀 Быстрый старт

### Требования

- Flutter SDK 3.10.4 или выше
- Dart 3.10.4 или выше
- Эмулятор Android/iOS или физическое устройство

### Установка

1. Перейдите в папку проекта:
```bash
cd mobile
```

2. Установите зависимости:
```bash
flutter pub get
```

3. Запустите приложение:
```bash
flutter run
```

### Запуск на конкретном устройстве

1. Посмотрите список доступных устройств:
```bash
flutter devices
```

2. Запустите на конкретном устройстве:
```bash
flutter run -d <device_id>
```

## 🔌 Интеграция с API

В проекте создан `ApiService` с методами-заглушками:

```dart
final apiService = ApiService();

// Поиск заболеваний по симптомам
final diseases = await apiService.searchSymptoms(['насморк', 'кашель']);

// Получение деталей заболевания
final disease = await apiService.getDisease(1);

// Получение методов лечения
final remedies = await apiService.getRemediesForDisease(1);

// Оценка метода
await apiService.rateRemedy(1, true, 'Отличный рецепт!');
```

Для подключения реального API:

1. Укажите `baseUrl` в конструкторе `ApiService`
2. Раскомментируйте HTTP-запросы в методах сервиса
3. Обновите модели при необходимости
